TIPOGRAFIA

  anatomia

    anatomia do tipo
    A b e g k n r s x Q

    altura das maiusculas = A Q
    ascendente = b k t d
    descendente = g q p j
    corpo = A b g k Q
    altura-x = x a m s

    serifa = pequeno traço nas letras em algumas fontes
      facilita a leitura no papel
    
    categoria de fontes
      serifada = TEM SERIFA
      sans-serif = SEM SERIFA (texto grande em sites)
      monoespaçada = todos os glifos tem o mesmo espaço (pode ou não ter serifa)
      handwriting/script = simula a escrita a mão
      display/comemorativas = fontes sem as caracteristicas a cima

MODELO DE CAIXAS

  aninhamento de CAIXAS

  altura = height
  largura = width

  linha que circunda o conteudo = border
  border ficar distante do conteudo = padding
  border ficar distante de OUTROS conteudos = margin
  contorno fora do elemento = outline

  padding = interno
  margin = externo

  tipos de CAIXAS
    box-level = QUEBRA A LINHA PARA ADICIONAR CAIXA E PREENCHE 100% DO WIDTH <div>
    inline-level = O CONTRARIO <span>
